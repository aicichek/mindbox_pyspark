# mindbox_pyspark

  func_pyspark.py Содержит в себе функцию, которая принимает в себя 3 датафрейма:
1) С продуктами
2) С Категориями
3) С связями продукт-категория

Первым шагом мы джоиним к датафрейму с продуктами датафрейм со связями

Далее мы группируем датафрейм по продукту и собираем все связанные категории в список

Далее мы к полученному датафрему присоединяем датафрейм с категориями и, при помощи селекта, оставляем в нем тольок названия кродуктов и категорий, без их id

Финальным шагом мы сортируем датафрейм по названия продуктов и возвращаем полученный датафрейм.


В test_func_pyspark.py я создал простые тестовые данные и запустил код для проверки работы
В результате получил датафрейм: 
+------------+-------------+
|product_name|category_name|
+------------+-------------+
|Product 1   |Category 1   |
|Product 1   |Category 2   |
|Product 2   |Category 2   |
|Product 3   |Category 3   |
|Product 4   |NULL         |
|Product 5   |Category 1   |
+------------+-------------+

Результат соответствует ожиданию.
